# React Form Actions – Practice

This project was built while learning React from the course  
**"React – The Complete Guide" by Maximilian Schwarzmüller (Academind)**.

It focuses on handling **form submissions using the form `action` approach**, including **asynchronous actions**, **optimistic updates**, and using `formStatus` for handling submission state.

---

## What I Practiced

- Handling forms using the `action` attribute
- Asynchronous form submissions
- Optimistic UI updates
- Managing `formStatus` state
- Input validation
- Handling user input in React
- Clean form handling patterns
